Quitar EXIF de una foto

Genera una copia nueva de la imagen sin el bloque EXIF original, directamente en tu dispositivo y sin depender de un servidor.

  • Procesado local en el navegador.
  • Descarga una copia limpia lista para compartir.
  • Confirmación visible de que la copia ya no conserva EXIF.

Cómo funciona

Al seleccionar la foto, la web la vuelve a generar localmente mediante el navegador para crear un nuevo archivo sin conservar el bloque EXIF incrustado en la imagen original.

Después verificamos la copia resultante para confirmar que ya no contiene EXIF legible.

Privacidad Tus fotos se procesan en tu navegador. No se suben a nuestros servidores.

Consejo útil

Si solo quieres saber si existe ubicación GPS antes de limpiar la foto, usa primero la herramienta específica para comprobar ubicación.
